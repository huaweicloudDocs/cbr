# 查询存储库资源实例<a name="ShowVaultResourceInstances"></a>

## 功能介绍

使用标签过滤实例 标签管理服务需要提供按标签过滤各服务实例并汇总显示在列表中，需要各服务提供查询能力

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=ShowVaultResourceInstances)中调试该接口。

## URI

POST /v3/\{project\_id\}/vault/resource\_instances/action
