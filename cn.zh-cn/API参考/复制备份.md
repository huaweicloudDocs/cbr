# 复制备份<a name="CopyBackup"></a>

## 功能介绍

跨区域复制备份。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=CopyBackup)中调试该接口。

## URI

{
  "replicate" : {
