# 查询存储库资源标签<a name="ShowVaultTag"></a>

## 功能介绍

查询指定实例的标签信息 标签管理服务需要使用该接口查询指定实例的全部标签数据

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=ShowVaultTag)中调试该接口。

## URI

GET /v3/\{project\_id\}/vault/\{vault\_id\}/tags
