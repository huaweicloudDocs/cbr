# 查询备份还原点<a name="ShowCheckpoint"></a>

## 功能介绍

根据还原点ID查询指定还原点

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=ShowCheckpoint)中调试该接口。

## URI

GET /v3/\{project\_id\}/checkpoints/\{checkpoint\_id\}
