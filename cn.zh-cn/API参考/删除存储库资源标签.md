# 删除存储库资源标签<a name="DeleteVaultTag"></a>

## 功能介绍

幂等接口：删除时，如果删除的标签不存在，返回404。Key不能为空或者空字符串。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=DeleteVaultTag)中调试该接口。

## URI

DELETE /v3/\{project\_id\}/vault/\{vault\_id\}/tags/\{key\}
