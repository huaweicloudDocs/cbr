# 查询复制能力<a name="ShowReplicationCapabilities"></a>

## 功能介绍

查询本区域的复制能力

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=ShowReplicationCapabilities)中调试该接口。
