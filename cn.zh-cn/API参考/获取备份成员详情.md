# 获取备份成员详情<a name="ShowMemberDetail"></a>

## 功能介绍

获取备份成员的详情

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=ShowMemberDetail)中调试该接口。

## URI

GET /v3/\{project\_id\}/backups/\{backup\_id\}/members/\{member\_id\}
