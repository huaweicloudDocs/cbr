# 添加备份成员<a name="AddMember"></a>

## 功能介绍

添加备份可共享的成员，只有云服务器备份可以添加备份共享成员，且仅支持在同一区域的不同用户间共享。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=AddMember)中调试该接口。

## URI

POST /v3/\{project\_id\}/backups/\{backup\_id\}/members
