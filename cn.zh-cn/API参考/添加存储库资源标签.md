# 添加存储库资源标签<a name="CreateVaultTags"></a>

## 功能介绍

一个资源上最多有10个标签。 此接口为幂等接口：创建时，如果创建的标签已经存在（key相同），则覆盖。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=CreateVaultTags)中调试该接口。

## URI

POST /v3/\{project\_id\}/vault/\{vault\_id\}/tags

## 请求示例

添加存储库标签

```
POST  https://{endpoint}/v3/{project_id}/vault/{vault_id}/tags

{
  "tag" : {
    "key" : "key1",
    "value" : "key2"
  }
}
```
