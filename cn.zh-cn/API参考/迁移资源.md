# 迁移资源<a name="MigrateVaultResource"></a>

## 功能介绍

支持资源迁移到另一个存储库，不删除备份。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=MigrateVaultResource)中调试该接口。

## URI

POST /v3/\{project\_id\}/vaults/\{vault\_id\}/migrateresources
