# 移除资源<a name="RemoveVaultResource"></a>

## 功能介绍

移除存储库中的资源，若移除资源，将一并删除该资源在保管库中的备份

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=RemoveVaultResource)中调试该接口。

## URI

POST /v3/\{project\_id\}/vaults/\{vault\_id\}/removeresources
