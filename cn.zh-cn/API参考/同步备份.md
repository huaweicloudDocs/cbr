# 同步备份<a name="ImportBackup"></a>

## 功能介绍

同步线下混合云VMware备份副本

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=CBR&api=ImportBackup)中调试该接口。

## URI

POST /v3/\{project\_id\}/backups/sync
